# Tech Interview Prep Bot

![Status](https://img.shields.io/badge/status-live-3b82f6) ![Gemini](https://img.shields.io/badge/AI-Gemini%202.0%20Flash-4285F4) ![HTML](https://img.shields.io/badge/built%20with-HTML%2FJS-orange)

> Agent #20 of 100 | 100 AI Agents Challenge

Simulate a real tech interview — coding, system design, or behavioral — and get instant per-answer feedback from an AI senior engineer. Ends with a full performance report and study recommendations.

## Features
- Coding, System Design, and Behavioral (STAR) modes
- Per-answer scoring with model answer shown
- Final grade + personalized improvement plan
- Tailored for FAANG, startups, internships, and new grad roles
- Question history saved locally for review
